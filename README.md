# 🧮 Simple C Calculator
A lightweight Command Line Interface (CLI) calculator that performs basic arithmetic operations.

## 📝 Description
This program takes an operator and two operands from the user and performs addition, subtraction, multiplication, or division. It includes error handling for division by zero and invalid operators.

## 🛠️ Concepts Covered
- **Switch-Case Statements**: For efficient decision-making logic.
- **Data Types**: Using `double` for high-precision decimal calculations.
- **Input Validation**: Handling division by zero errors.
- **Formatted Output**: Using `%.2lf` to limit decimal places for cleaner results.

## 🚀 How to Run
1. Compile the code: `gcc calculator.c -o calculator`
2. Run the program: `./calculator`
